```js
var hammingWeight = function(n) {
       let ret = 0;
       while (n) {
        n &= n - 1;
        ret++;
    }
    return ret;
}
```

由于位运算符&会在运算之前系统自动将数字转换成32位的二进制数，所以不用自行转换，且固定n&n-1会使n少去一个1，所以只要n不为全零就一直循环，n每&n-1一次，就少一个1，计数器就加1，当n为全零没有1时，就退出得出1的个数
